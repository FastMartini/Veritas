# Veritas v0.5.0 - Settings

## Overview
Version 0.5.0 introduces a dedicated settings page that allows users to control how analysis results are displayed. This improves usability by giving users direct control over what information they see without cluttering the main popup interface.

## Key Changes

### Dedicated Settings Page
- Introduced `settings.html` as a standalone interface for user preferences.
- Navigation flow:
  - Popup → Settings via settings button
  - Settings → Popup via back button

## Features

### 1. Toggle On/Off Dark Mode
### 2. Toggle On/Off Article Overview
### 3. Toggle On/Off Extracted Claims
### 4. Toggle On/Off Extraction text box
